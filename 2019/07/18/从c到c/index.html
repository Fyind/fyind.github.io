枚举

可以一次性生成一系列的递增的值，如果没有复制的话，后面一个总是比前面一个多1

  • 枚举变量不能赋值
  • 不同类型的枚举变量不能复制

用处

可以在for循环里面枚举

可以表示多个同一个类型的数据

内联

函数前面加 inline.

可以加快简单函数的执行速度

重载

同一作用域，函数名相同，参数表不同的函数构成重载

默认参数，哑元

定义默认参数，必须从右边开始设置

哑元用于兼容老版本的函数

调用的时候都是可以些3个参数

引用
